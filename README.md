# 🚀 DevTrack API

A simple backend API built with FastAPI for user management and authentication.

---

## 📌 Features
- User registration
- User login (JWT)
- CRUD operations
- Protected routes
- Password hashing

---

## 🛠️ Tech Stack
- Python
- FastAPI
- SQLAlchemy
- JWT
- SQLite
